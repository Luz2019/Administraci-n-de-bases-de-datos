Objetivos del OVI

Objetivos general del OVI

Desarrollar un sitio web en formato HTML5 y CSS3, para ser visualizados en diferentes navegadores teniendo en cuenta su funcionabilidad y usabilidad, por medio del uso de diferentes editores de código propuestos en el curso.

Objetivos específicos del OVI

Estudiar las temáticas relacionadas con los referentes del diseño web, Sistemas de control de versiones y Editores de código para la web. Seleccionar el curso del programa Tecnología en Desarrollo de Software para el desarrollo de un sitio web que refleje los conocimientos adquiridos. Aplicar los conceptos para realizar una maquetación o Mockup de la página Index del sitio web del curso seleccionado.
